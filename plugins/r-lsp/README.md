# r-lsp

R language server for Claude Code, providing code intelligence for `.R`, `.Rmd`, and `.qmd` files.

## Prerequisites

Install the [`languageserver`](https://github.com/REditorSupport/languageserver) R package:

```r
install.packages("languageserver")
```

## Installation

```
/plugin marketplace add christopherkenny/skills
```

Then install `r-lsp` from the marketplace.
