# PL94171 — Census Decennial Data

## Overview

`PL94171` ingests Census P.L. 94-171 redistricting data — the official data file used for redistricting after each decennial Census. It provides block-level population and race/ethnicity counts required to meet equal-population and Voting Rights Act standards.

## Loading Census Data

```r
library(PL94171)

# Download P.L. 94-171 data for a state
md_data <- pl_raw('MD', year = 2020)   # 2020 Census redistricting file

# Process into a tidy format
md_tidy <- pl_tidy(md_data)
```

## Key Tables

The P.L. 94-171 file has three tables:

| Table | Contents |
|-------|---------|
| P1 | Race alone (total population) |
| P2 | Hispanic or Latino + race (total population) |
| P3 | Race alone (18+ voting-age population) |
| P4 | Hispanic or Latino + race (18+ voting-age population) |

```r
# Access specific tables
p1 <- pl_tidy(md_data, table = 'P1')   # total population by race
p3 <- pl_tidy(md_data, table = 'P3')   # VAP by race
```

## Standard Population Variables

After processing with `pl_tidy()`:

| Variable | Description |
|----------|-------------|
| `pop` | Total population |
| `pop_hisp` | Hispanic/Latino (any race) |
| `pop_white` | Non-Hispanic White alone |
| `pop_black` | Non-Hispanic Black/AA alone |
| `pop_aian` | Non-Hispanic American Indian/Alaska Native |
| `pop_asian` | Non-Hispanic Asian alone |
| `pop_nhpi` | Non-Hispanic Native Hawaiian/Pacific Islander |
| `pop_other` | Non-Hispanic other/multiracial |
| `vap` | Voting-age population (18+) |
| `vap_hisp` | Hispanic/Latino VAP |
| `vap_white` | Non-Hispanic White VAP |
| `vap_black` | Non-Hispanic Black VAP |
| (etc.) | Same race breakdown for VAP |

## Aggregating Blocks to Precincts

Census blocks must typically be aggregated to the precinct or VTD (voting tabulation district) level for redistricting analysis:

```r
library(tidyverse)
library(PL94171)
library(sf)

# 1. Load block data with geometry
blocks <- pl_raw('MD', 2020, level = 'block', geometry = TRUE)
blocks <- pl_tidy(blocks)

# 2. Load precinct shapefile
precincts <- st_read('path/to/precincts.shp')

# 3. Match blocks to precincts by centroid
block_assignments <- geo_match(st_centroid(blocks), precincts)

# 4. Aggregate population to precincts
precinct_pop <- blocks |>
  mutate(precinct = block_assignments) |>
  group_by(precinct) |>
  summarize(across(c(pop, pop_black, pop_hisp, vap, vap_black), sum))

# 5. Join to precinct shapefile
precincts <- precincts |>
  left_join(precinct_pop, by = ...)
```

## VTDs (Voting Tabulation Districts)

The Census also provides VTDs, which roughly correspond to precincts:

```r
vtds <- pl_raw('MD', 2020, level = 'vtd', geometry = TRUE)
vtds <- pl_tidy(vtds)
```

VTDs are often the best starting point for precinct-level data since they are designed to align with election administration boundaries.

## Building a Custom `redist_map`

library(redistverse)

map <- redist_map(
